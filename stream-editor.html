Stream Editor

C Programming & Systems Development

CSC 322 Midterm Project • File I/O • String Manipulation • Data Structures

A command-line stream editor written in C that processes text input according to edit commands specified in a file. Similar to the UNIX sed utility, it supports line range specifications, text substitution, insertion, deletion, and appending operations.

C Programming File I/O String Processing Data Structures

Edit Operations

  • • Append (A): Add text to end of line
  • • Insert (I): Add text to start of line
  • • Output (O): Insert new line before current
  • • Substitute (s): Replace text patterns
  • • Delete (d): Remove entire lines

Line Range Specifications

  • • All lines: No specification
  • • Text match: /pattern/ format
  • • Line numbers: start,end/ format
  • • Flexible targeting: Apply edits selectively

Technical Implementation

Data Structures:

  • • Structs for edit commands
  • • Unions for line range data
  • • Enums for specification types
  • • Arrays for command storage

Core Features:

  • • Standard I/O processing
  • • Command file parsing
  • • Pattern matching with strstr()
  • • Memory-safe string operations
